# Vimac

- Tap alone тЖТ `Escape`
- Hold тЖТ navigation layer

## Window Tiling (Rectangle)

Two modes. Same keys inside both.

| Trigger | Mode | Exits when |
| :--- | :--- | :--- |
| `Caps Lock + t` | One-shot | After one action |
| `Caps Lock + T` | Toggle | `Caps Lock` alone |

**Halves:**

| Key | тЖТ |
| :---: | :--- |
| `H` | Left half |
| `L` | Right half |
| `K` | Top half |
| `J` | Bottom half |

**Quarters** (spatial layout):

```
Y  U   тЖТ  top-left   top-right
B  N   тЖТ  bottom-left  bottom-right
```

| Key | тЖТ |
| :---: | :--- |
| `Y` | Top-left |
| `U` | Top-right |
| `B` | Bottom-left |
| `N` | Bottom-right |

**Other:**

| Key | тЖТ |
| :---: | :--- |
| `Space` | Maximize |
| `R` | Restore |
