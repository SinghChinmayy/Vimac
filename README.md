# Vimac

Vim-style navigation on macOS via Karabiner-Elements. `Caps Lock` is the super-key.

- Tap alone ‚Üí `Escape`
- Hold ‚Üí navigation layer

---

## Desktop Navigation

Hold `Caps Lock` + key:

| Key | ‚Üí |
| :---: | :--- |
| `H` | ‚Üê |
| `J` | ‚Üì |
| `K` | ‚Üë |
| `L` | ‚Üí |
| `U` | Home (line start) |
| `I` | End (line end) |
| `F` | Word ‚Üê |

## Window Tiling (Rectangle)

Two modes. Same keys inside both.

| Trigger | Mode | Exits when |
| :--- | :--- | :--- |
| `Caps Lock + t` | One-shot | After one action |
| `Caps Lock + T` | Toggle | `Caps Lock` alone |

**Halves:**

| Key | ‚Üí | Rectangle shortcut |
| :---: | :--- | :--- |
| `H` | Left half | `^‚å•‚Üê` |
| `L` | Right half | `^‚å•‚Üí` |
| `K` | Top half | `^‚å•‚Üë` |
| `J` | Bottom half | `^‚å•‚Üì` |

**Quarters** (spatial layout):

```
Y  U   ‚Üí  top-left   top-right
B  N   ‚Üí  bottom-left  bottom-right
```

| Key | ‚Üí | Rectangle shortcut |
